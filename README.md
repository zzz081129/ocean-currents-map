# 🌍 Ocean Currents & Marine Geography Visualization

## 🇨🇳 中文介绍

这是一个用于地理教学的可视化演示系统（基础版），专注于展示全球海洋与地理相关知识。

### ✨ 功能特点

- 🌊 展示全球主要洋流分布
- 🗺 标注世界知名海域名称
- 📚 集成基础地理知识点
- 🔧 支持自由扩展与内容添加（可自定义数据）

### 🎯 项目定位

本项目适用于：

- 地理教学辅助
- 学习演示工具
- 可视化探索

---

## 🇬🇧 English Description

This is a basic version of a geography teaching visualization system, designed to present global ocean and marine-related knowledge in an interactive way.

### ✨ Features

- 🌊 Displays major global ocean currents
- 🗺 Shows names of well-known seas and ocean regions
- 📚 Includes basic geography knowledge points
- 🔧 Supports flexible extension and custom content

### 🎯 Use Cases

- Geography education
- Interactive learning tools
- Visualization projects

---

## 🇩🇪 Deutsche Beschreibung

Dies ist eine grundlegende Version eines Visualisierungssystems für den Geographieunterricht, das globale Meeres- und Geographiedaten interaktiv darstellt.

### ✨ Funktionen

- 🌊 Darstellung der wichtigsten globalen Meeresströmungen
- 🗺 Anzeige bekannter Meeres- und Ozeangebiete
- 📚 Integration grundlegender geographischer Inhalte
- 🔧 Erweiterbar und anpassbar mit eigenen Daten

### 🎯 Anwendungsbereiche

- Geographieunterricht
- Interaktive Lernwerkzeuge
- Visualisierungsprojekte
